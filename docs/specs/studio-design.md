# Studio — редизайн PromptW WebApp (спека)

Направление: **«Studio»** — тёплый тёмный + цветовое кодирование зон. Современный,
драйвовый, премиальный, **не «AI-дефолт»**, аудитория 25–40. Это **рестайл существующей
структуры**, а не реинвенция: секции и функции остаются, меняются палитра, типографика,
геро и акценты.

## Принципы
- Тёплая тёмная база (НЕ сине-чёрная) → живо, не мрачно, не «AI».
- Яркому цвету — площадь (сочные тайлы/акценты), а не тонкие неон-линии.
- **Всё плоско, ноль свечения/градиент-glow** (это и отличает от AI). Градиент допустим
  только тёплый и редко (бренд/баланс), не холодный violet→cyan.
- **Цвет = навигация:** каждый раздел владеет цветом (фото/видео/аудио/чат).
- Главная = **витрина трендов/шаблонов сразу** (вау-эффект). Ядро продукта — шаблоны
  («загрузил фото → вау-результат»), ручной промпт («С нуля») вторичен.

## Токены (CSS :root)
```
--bg:    #16130F   тёплый уголь (экран)
--bg2:   #1E1A15   приподнятый фон
--card:  #241F19   карточки
--card2: #2E2820   инпуты/шторки/тосты
--line:  #3A332A   границы/делители
--tx:    #F5F0E8   основной текст
--tx2:   #B5AC9E   вторичный
--tx3:   #7C7565   подсказки/лейблы

Зоны (цвет = навигация):
--photo:  #FF6B5C  фото / главный путь
--video:  #FFB23E  видео
--audio:  #3DD68C  аудио
--chat:   #8C9BFF  чат/текст
--gold:   #FFCB6B  баланс / пополнить

Тёплый бренд-градиент (редко): linear-gradient(135deg,#FF6B5C,#FFB23E)
Радиусы: --rs:12 · --r:16 · шторки 20 · пилюли 999
```
Текст на ярких плашках — тёмный из той же гаммы (напр. на коралле `#3A1109`), не чёрный.

## Типографика (3 роли)
- **Display — Unbounded** (Google Fonts, 500/700): заголовки, геро, названия разделов,
  крупные числа. Характерный, премиум-драйв.
- **Body — Manrope** (уже подключён): тело, описания, UI-текст.
- **Mono — Space Mono** (Google Fonts): баланс, цены (W), статы, микро-лейблы (вместо
  псевдо-моно на Manrope).
Иерархия весом+гарнитурой, не размером. tnum включён.

## По экранам (рестайл, структура сохраняется)
- **Главная:** welcome-баннер → компактный хук «Загрузи фото — получи вау-результат» +
  «По шаблону» (коралл, primary) и «С нуля» (вторично). Дальше сразу **Тренды** (карточки
  результатов, моно-бейдж типа), затем **Шаблоны** (категории с заполненными превью).
  Убрать звёздо-маскот, точки-карусель, декоративные бары переосмыслить.
- **Создать:** форма красится в цвет выбранного типа (таб ФОТО=коралл / ВИДЕО=амбер /
  АУДИО=нефрит). Кнопка генерации — в цвете зоны. Сег-переключатель типов — на новой палитре.
- **Текст/Чат:** акцент перивинкл (`--chat`), пузыри/инпут/кнопка send.
- **История:** плитки результатов; компактный ряд действий под медиа уже есть (Сохранить/
  Повторить/Поделиться) — перекрасить под Studio.
- **Профиль:** баланс-карта в золоте, плитки действий, партнёрка — тёплый люкс.
- **Оплата:** шторка/пакеты на новой палитре, «Пополнить» в золоте.

## Реализация (инкрементально)
1. Глобальные токены + подключение шрифтов (рекрасит весь апп консистентно).
2. Главная.
3. Создать → История → Чат → Профиль → Оплата.
Каждый шаг: бамп `?v=N`, деплой на ревью. Сохранять: reduced-motion, фокус-стейты,
i18n-паритет, все функции. Кастомные line-иконы, без эмодзи.
